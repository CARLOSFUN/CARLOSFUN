# Crosscheck

Crosscheck is a Microsoft Defender for Endpoint threat hunt focused on a year end insider threat scenario in a controlled lab environment.

## Repository Contents

* SOC style hunt report
* KQL library for Flags 01 to 22
* Attack timeline template
* Screenshot placeholders and naming guidance

## How to Use

1. Run each KQL file in Microsoft Defender for Endpoint Advanced Hunting.
2. Capture supporting screenshots and name them using the convention in `screenshots/README.md`.
3. Update the report and timeline placeholders with confirmed findings and evidence.

## Disclaimer

All content is based on a sanitized lab environment. No real organization data is included.
